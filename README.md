# 💳 Vault — Premium Banking App

A Streamlit-based personal banking application.

---

## Requirements

- Python 3.10 or higher
- pip

---

## Setup & Run

**1. Clone the repository**
```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

**2. Install dependencies**
```bash
pip install streamlit
```

**3. Run the app**
```bash
streamlit run bank_app.py
```

**4. Open in browser**

Streamlit will automatically open the app at:
```
http://localhost:8501
```

---

## Notes

- Account data is saved locally in `data.json` (auto-created on first use).
- Keep `bank_app.py` and `data.json` in the same directory.
